## ✨ 简介 (Introduction)

这是一个极具未来感与科技风的个人主页项目，主题为“**智算无界**”。网站采用了赛博朋克风格（Cyberpunk）的深色主题，辅以青色和紫色的霓虹灯效，全方位展示了开发者的个人介绍、教育背景、实习经历、重大科研项目以及精美的作品集。

## 🚀 特色功能 (Features)

- **🌌 沉浸式互动背景**: 使用 `tsParticles` 打造了动态跟随鼠标的粒子连线特效。
- **🎬 流畅的滚动动画**: 基于 `Framer Motion`，每一个模块、卡片在滚动进入视口时都有优雅的渐显与悬浮动画。
- **📱 全响应式设计**: 深度使用 `Tailwind CSS`，完美适配桌面端、平板及移动端。
- **🧩 瀑布流画廊**: 使用自定义 Masonry 布局展示生活与风采照片，打破传统网格的死板。
- **⚡ 极速构建与部署**: 采用 `Vite` 作为构建工具，配合 GitHub Actions 实现全自动 CI/CD 部署到 GitHub Pages。

## 🛠️ 技术栈 (Tech Stack)

- **框架**: [React 18](https://react.dev/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **样式与排版**: [Tailwind CSS](https://tailwindcss.com/)
- **动画引擎**: [Framer Motion](https://www.framer.com/motion/)
- **粒子特效**: [tsParticles](https://particles.js.org/)
- **图标库**: [Lucide React](https://lucide.dev/)
- **部署**: GitHub Pages (Automated via GitHub Actions)

## 📦 本地运行 (Local Development)

克隆项目到本地后，使用 npm 进行依赖安装及运行：

```bash
# 1. 克隆项目
git clone https://github.com/AcKingW/AcKingW.github.io.git

# 2. 进入目录
cd AcKingW.github.io

# 3. 安装依赖
npm install

# 4. 启动本地开发服务器
npm run dev
```

运行成功后，在浏览器中打开 `http://localhost:5173`（具体端口以终端输出为准）即可预览。

## 🏗️ 生产构建 (Build for Production)

```bash
# 打包生成生产环境的静态文件
npm run build

# 本地预览打包后的效果
npm run preview
```
